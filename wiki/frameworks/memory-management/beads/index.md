# Beads: система памяти для кодинг-агентов

## Что есть

Система управления памятью и поведением кодинг-агентов, которая упрощает взаимодействие с LLM.

## Почему важно

- Позволяет искусственному интеллекту:
  - Память деталей кода
  - Соблюдать архитектурные принципы
  - Сохранять согласованность репозитория
