# Archive
Архив исходных материалов после обработки и интеграции в wiki.

## Что здесь
- Оригиналы статей, транскриптов, скриншотов, PDF из `raw/`
- Обработанные вырезки из `Clippings/`
- Файлы перемещаются сюда **после** успешной интеграции знаний в `wiki/`

## Структура
Повторяет структуру источников:
```
archive/
├── raw/
│   ├── articles/
│   ├── transcripts/
│   ├── screenshots/
│   └── ...
└── Clippings/
```

## Политика хранения
- Минимальный срок: **90 дней**
- По истечении — можно сжимать или удалять при нехватке места
- Старые архивы группируются по годам: `archive/2024/raw/`

## Зачем это нужно
- Сохранение исходников для аудита и верификации
- Возможность переработать материал при изменении стандартов
- Защита от потери данных
