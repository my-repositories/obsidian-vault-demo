# Clippings
Веб-вырезки, быстрые сохранения и фрагменты материалов, ожидающие обработки.

## Что сюда попадает
- Вырезки из статей, блогов, документации
- Фрагменты кода и сниппеты
- Короткие заметки и идеи, сохранённые на лету
- Фрагменты обсуждений (форумы, Reddit, Habr)

## Отличие от raw/
- **`raw/`** — цельные материалы: полные статьи, PDF, транскрипты, скриншоты
- **`Clippings/`** — фрагменты, вырезки, короткие сохранения без полного контекста

## Жизненный цикл
1. Файл попадает в `Clippings/`
2. При ревизии обрабатывается: извлекаются ключевые идеи
3. Знания интегрируются в `wiki/`
4. Оригинал переносится в `archive/Clippings/`

## Формат файлов
- Markdown, TXT, или любой читаемый формат
- Именование: `YYYY-MM-DD-краткое-описание.ext`
