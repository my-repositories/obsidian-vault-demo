# База знаний по вайбкодингу
Это стартовая структура Obsidian-хранилища для сбора и систематизации знаний по вайбкодингу.

## Структура
- [raw/readME.md](raw/readME.md.md) — сырые материалы: статьи, заметки, скриншоты, PDF.
- [wiki/readME.md](wiki/readME.md.md) — структурированные wiki-страницы по темам.

## Принцип работы
1 Всё входящее сначала складывается в `raw/`.
2 Затем материалы разбираются, очищаются и превращаются в wiki-страницы в `wiki/`.
3 По каждой крупной теме в `wiki/` создавается отдельная папка со своим `readME.md`.

## Темы для постинга
- AI IDE
- Cursor
- Windsurf
- Claude Code
- Prompts
- Agents
- MCP
- Tools
- Workflows
- Кейсы
- Ошибки и анти-паттерны

## Правила
- Не смешивай сырые заметки и итоговые wiki-материалы.
- Для новых тем создавай отдельные подпапки внутри `wiki/`.
- Внутри каждой темы должен быть свой `readME.md` как точка входа.
